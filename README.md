# 工程文件夹说明

|路径|说明|
|---|---|
|./src/directives/|自定义指令目录|
|./src/components/|自定义组件目录|
|./src/pages/|前端页面|
|./src/static/|静态资源|
|./src/store/|Vuex全局状态|
|./src/utils/|项目使用到的API文件夹|
|./src/App.vue|程序的生命周期钩子函数|
|./src/main.js|用来添加全局引用的模块|
|./package.json|npm依赖|
|./src/pages.json|路由|
|./src/uni.scss|全局样式|
|./.env|全局静态常量|
|./vite.config.js|Vite编译配置文件|
|./start.ps1|快速启动脚本|

# 使用教程
右键`start.ps1`点击使用 powershell 运行，选 1 然后选 2 .
